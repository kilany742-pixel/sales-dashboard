# 📊 لوحة تحكم المبيعات — Sales Dashboard

لوحة تحكم تفاعلية احترافية لتحليل بيانات مبيعات منطقة العاشر من رمضان.

## 🚀 عرض مباشر

افتح `index.html` في أي متصفح — لا يحتاج سيرفر أو إعدادات.

## ✨ المميزات

| الميزة | التفاصيل |
|--------|---------|
| 📱 Responsive | يعمل بشكل مثالي على الموبايل واللاب |
| ✨ Animations | انيميشن سلس على الكروت والأرقام والشارتات |
| 🌙 Dark Theme | تصميم داكن احترافي |
| ↔️ RTL Support | دعم كامل للعربية من اليمين لليسار |
| 📊 5 صفحات | رئيسية / مناطق / عملاء / أصناف / آخر 10 أيام |

## 📂 الصفحات

- **🏠 الرئيسية** — KPIs + دونات المناطق + أكبر العملاء + توزيع العملاء
- **🗺️ المناطق** — مقارنة المبيعات + أقل العملاء + أكثر الفواتير  
- **👥 العملاء** — أكبر 10 عملاء + أكثر صنف يشتريه كل عميل
- **📦 الأصناف** — أكبر 5 أصناف في كل منطقة مع progress bars
- **⏱️ آخر 10 أيام** — أكبر العملاء + بلا مبيعات + مبيعات بالمنطقة + الفواتير
